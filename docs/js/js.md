## 输出语句
```js
console.log() //控制台输入
window.alert() //弹出警告
```

## 数据类型
```基本类型:(String)字符串, (Number)数字, (boolean)布尔, (Null)空, (Undefined)未定义 ```

```引用类型:(Object)对象, (Array)数组, (Function)函数```
```js
var length = 7;  // 数字
var lastName = "Gates";  // 字符串
var cars = ["Porsche", "Volvo", "BMW"];  // 数组
var x = {firstName:"Bill", lastName:"Gates"}; // 对象 
```
