## ajax的请求过程
```创建 XMLHttpRequest 对象,也就是创建一个异步调用对象```

```创建一个新的 HTTP 请求,并指定该 HTTP 请求的方法、URL 及验证信息```

```设置响应 HTTP 请求状态变化的函数```

```发送 HTTP 请求```

```获取异步调用返回的数据```

```使用 JavaScript 和 DOM 实现局部刷新```

## git 和 post 的区别

```传输大小:git请求一般只能传输2kb,post传输参数默认不受限制```

```git请求可以在浏览器上看到穿的参数不安全 ,post 请求看不到穿的参数所以他比git请求安全 ```

```post 请求无法缓存```
